# Screenshots

Drop real iPhone screenshots here with these filenames — the site picks them up
automatically (until then it shows a striped placeholder phone):

- `chat.png`   — the main chat answering a family question
- `setup.png`  — the conversational "Meet Frankly" setup
- `voice.png`  — the voice conversation screen (the orb)
- `files.png`  — Files & folders with a filed photo

Take them on a real device (Volume Up + Side button) or in the Simulator
(File ▸ New Screen Shot, ⌘S). PNG, portrait. Roughly 1170×2532 (iPhone) is ideal;
the site scales them.
